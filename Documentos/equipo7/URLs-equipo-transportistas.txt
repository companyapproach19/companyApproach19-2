URL: "/recogidoPedido"
	El metodo recogidoPedido recibe un String llamado json (formato JSON firmaRecogida), 
	que es el JSON del pedido con la firma en la recogida del pedido y el actor que realiza el transporte
	Devuelve un String que es el JSON del pedido con el estado cambiado

URL: "/entregadoPedido"
	El metodo entregadoPedido recibe un String llamado json (formato JSON transportistas_entrega), 
	que es el JSON del pedido con la firma en la entrega del pedido y los datos del registro
	Devuelve un String que es el JSON del pedido con el estado cambiado


En caso de dudas, consultar src/main/java/com.controller/ManejaPeticiones.java
El formato de los JSON se encuentra en Documentos/Equipo7
